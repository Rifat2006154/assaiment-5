Question-2: What is the difference between innerHTML, innerText, and textContent ?

When we use InnerHTML it's include full html structure, innerText take only text content of an element, textContent take all text content including hidden elements

Question-3: What is event delegation in the DOM?

Event delegation হল JavaScript-এর একটি কৌশল যেখানে parent element-এর উপর event listener সেট করা হয়, যাতে তার child elements-গুলোর event handle করা যায়। এটি বিশেষভাবে কার্যকর যখন dynamicভাবে element তৈরি হয় বা প্রচুর child element থাকে।
